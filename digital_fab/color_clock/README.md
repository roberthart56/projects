PLA parts to position the stepper motor.  Slots allow adjustment in two directions.  Captured nuts for connecting parts and attaching spindle to motor shaft.


| | | | |
|---|---|---|---|
| ![1](./img/assembly.jpg) | ![1](./img/parts.jpg) | ![1](./img/nuts.jpg) | ![1](./img/holder.jpg)  |

Electronics for the stepper are on a modular board from MIT.  The board fits into a case.  An OLED is used for display, and buttons change the stepper period.  First prototype shown below, in which a breadboard with messy wires is used to develop software.
